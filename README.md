# Gravatar::Rails

Gravatars for the modern web. This Gem is coming soon. Right now, it is pretty much an empty gem project.

## Installation

Add this line to your application's Gemfile:

    gem 'gravatar-rails'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install gravatar-rails
